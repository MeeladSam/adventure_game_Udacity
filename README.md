# Adventure Game

This is a simple text-based adventure game made with Python.

In this game, the player explores places, makes choices, and tries to win.
Some parts of the game change each time because of randomness.

## Features

- Print story text to the screen
- Ask the player to choose between options
- Check for invalid input
- Random enemy or item
- Win or lose ending
- Play again option

## How to run

Make sure Python is installed, then run:

```bash
python adventure_game.py
